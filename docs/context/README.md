# Contexto da Sessao

Este diretorio guarda o contexto operacional do projeto para retomar o trabalho sem depender do historico do terminal.

## Arquivos

- `docs/context/CURRENT_STATE.md`: snapshot do estado atual do repositorio e itens em andamento.
- `docs/context/RECEIVABLES_HANDOFF.md`: handoff detalhado do modulo de Receivables (UI, UX e estado atual).
- `docs/context/WORKLOG_RECEIVABLES.md`: log incremental de mudancas realizadas no modulo.
- `docs/context/RESUME_CHECKLIST.md`: passos rapidos para encerrar e retomar o trabalho com seguranca.

## Uso recomendado

1. Antes de fechar o terminal, atualizar `docs/context/CURRENT_STATE.md` com o que mudou no dia.
2. No retorno, seguir `docs/context/RESUME_CHECKLIST.md`.
3. Para continuar front-end de financas, usar `docs/context/RECEIVABLES_HANDOFF.md` como fonte principal.
